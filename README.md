# ELEO – Fashion Discovery Platform

ELEO is a web platform that helps users discover outfit inspiration and connect with verified fashion professionals across Nigeria.

The platform combines an intelligent fashion assistant with a growing directory of verified tailors, making it easier for customers to find style inspiration and trusted fashion services in one place.

## Features
- AI-powered outfit inspiration and fashion recommendations
- Verified tailor directory
- Search tailors by location and specialization
- Individual tailor profile pages
- Direct connection to tailors through WhatsApp and social media
- Responsive web design
- Admin approval system for tailor registrations
- Airtable-powered data management

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Airtable
- Netlify
- Botpress (Conversational AI Assistant)

## Live Website
https://eleostyleengine.netlify.app

## Purpose
ELEO was developed to simplify fashion discovery by combining AI-assisted outfit recommendations with a trusted directory of verified fashion professionals. The platform helps customers discover styles they love while making it easier to connect with skilled tailors who can bring those ideas to life.
